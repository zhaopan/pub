# foo | 高频 Git 命令清单

> 本文档为作者个人最常用的 Git 命令集合，具备高执行频率、易于记忆的特点，建议设置 alias 快捷方式使用。

---

## ✅ 初始化 & 基础操作

```bash
git init
git clone <repo>
git status
git add .
git commit -m "commit message"
git push
git pull
```

---

## ✅ 分支操作

```bash
git checkout -b dev
git checkout main
git merge dev
git rebase main
git branch -d dev
```

---

## ✅ 日常调试命令

```bash
git log --oneline --graph --all
git diff
git stash / stash pop
git reset --hard HEAD~1
git revert <commit>
```

---

## ✅ 常用 alias 建议

```bash
git st       # status
git co       # checkout
git ci       # commit
git br       # branch
git lg       # log --oneline --graph
```
