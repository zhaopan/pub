温馨提示：下载安装包最好不要下载最新版，下载比最新版底1-2个版本
==================================================================================================================
1)、首先安装jdk环境
==================================================================================================================
2)、安装 apache 【最好下载安装版的，绿色版的使用命令安装：使用cmd命令切换到bin目录下运行httpd.exe -k install】
找到安装目录下的/conf/httpd.conf打开修改
a、修改端口：Listen 80
b、修改 IfModule dir_module
<IfModule dir_module>
    DirectoryIndex index.html index.php
</IfModule>
c、修改站点目录 DocumentRoot【默认是在安装目录的htdocs目录下】
DocumentRoot "d:/PHPProject"
<Directory "d:/phpproject"> 修改目录权限
保存重启：输入地址测试
==================================================================================================================
3)、配置 php
a、修改 apache /conf/httpd.conf文件
LoadModule php5_module D:/php/php-5.5.5/php5apache2_4.dll
PHPIniDir "D:/php/php-5.5.5"
AddType application/x-httpd-php .php .phtml
b、修改 php 目录下的 php.ini 【默认没有需要把 php.ini-development 文件复制一份修改文件名 php.ini 】
extension_dir = "D:/php/php-5.5.5/ext"
==================================================================================================================
4)、mysql安装【省略】
==================================================================================================================
5)、mysql与php
a、修改 php 目录下的 php.ini 文件启用mysql
【这几个取消注释即可，根据php版本来使用mysql.dll】
extension=php_pdo_mysql.dll【php5.5不支持了】
extension=php_pdo_mysqli.dll
extension=php_pdo_mysql.dll
==================================================================================================================
6)、CI框架配置
a、去掉url中的index.php
在跟目录创建文件 .htaccess【注意必须和index.php同目录】
内容如下：
RewriteEngine on
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond $1 !^(index\.php|images|robots\.txt)
RewriteRule ^(.*)$ /PhpProject1/index.php/$1 [L]
修改 apache 中 httpd.conf 文件
LoadModule rewrite_module modules/mod_rewrite.so【取消注释】
用htaccess关键字收索
在 <Directory "D:/PHPProject"> 节点内修改
AllowOverride None==>AllowOverride All
重启服务 OK！
